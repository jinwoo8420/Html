구분 표현 설명
전체 선택자 * 모든 태그에 적용
태그 선택자 태그명 해당 태그에 적용
아이디 선택자 #아이디명 해당 id가 지정된 태그에 적용
유일해야한다
클래스 선택자 .클래스명 해당 class가 지정된 태그에 적용
동일한 클래스를 여러 태그에 선언가능
자식 선택자 선택자 > 자식 선택자 선택자의 바로 아래 위치한 자식 선택자에게만 적용
자손 선택자 선택자 자손 선택자 선택자의 아래 위치한 모든 자손 선택자에게 적용
(자식도 포함)
속성 선택자 선택자 [속성 = 값] 속성이 특정 값을 갖는 선택자에게 적용

텍스트 크기 확인

전체 선택자를 통해 html 파일 내의 모든 태그에 대해 동일한 스타일 적용 가능
