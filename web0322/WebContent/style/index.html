style 지정

인라인 스타일 태그에 직접 스타일 지정 <태그 style = '속성명 : 값; 속성명2 : 값;'>
내부 스타일 head 태그 안에 style 태그 선언하고
style 태그 내부에 스타일 지정		 <style>선택자 {속성명1 : 값; 속성명2 : 값;}
외부 스타일 외부 파일에 스타일을 지정하고
head 태그 안에 외부 파일을 연결하기 위한 링크 선언		 <link rel = 'stylesheet' href = '스타일 파일의 경로'>
