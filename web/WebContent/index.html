1.블록 레벨 텍스트 태그 : 혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시



1) h (heading) : 제목을 표시하는 태그 h1~6

가장 큰 제목 태그

가장 작은 제목 태그


2) p (paragraph) : 단락 표시로 줄바꿈 없이 한 줄로 표시
텍스트의 길이가 브라우저 창의 너비보다 길면 자동으로 줄 바뀜



3) br : 줄바꿈 태그



4) hr (horizontal) : 주제가 바뀌거나 분위기 전환을 위한 가로선 표시



5) div (division) : 영역을 나누는 블록 태그



2.인라인 레벨 텍스트 태그 : 한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠 만큼의 너비를 차지하는 태그



1) strong 태그 : 텍스트를 굵게 표시

2) span 태그 : 특별한 뜻은 없지만 텍스트의 일부만 스타일 적용하고자 할 때 사용
텍스트 일부만 스타일 적용


3. 목록을 만드는 태그



1) ul (unordered list) : 순서없는 목록 태그
기본 disc : list - style - type
  • 메뉴1
  • 메뉴2
  • 메뉴3
  • 메뉴4
  • 메뉴5


2) ol (ordered list) : 순서가 있는 목록 태그 (숫자, 알파벳, 로마숫자로 표시)
시작값 지정 가능 : start
설정 변경 : type
기본 decimal : list - style - type
  1. 메뉴1
  2. 메뉴2
  3. 메뉴3
  4. 메뉴4
  5. 메뉴5
